### Значения по умолчанию

Если не инициализировать поле явно, оно получит значение по умолчанию:

| Тип | Значение по умолчанию |
|-----|----------------------|
| `int`, `long`, etc. | 0 |
| `double`, `float` | 0.0 |
| `boolean` | false |
| `char` | '\u0000' |
| Объекты (String, etc.) | null |



### Поля vs локальные переменные

| Характеристика | Поля | Локальные переменные |
|----------------|------|---------------------|
| Где объявлены | В классе | В методе |
| Область видимости | Весь объект | Только метод |
| Значение по умолчанию | Есть | Нет (обязательная инициализация) |
| Время жизни | Пока живёт объект | Пока выполняется метод |



### Когда использовать static?

| Используй static | Не используй static |
|------------------|---------------------|
| Утилитные методы (Math.abs) | Методы, работающие с состоянием объекта |
| Константы (Math.PI) | Данные, уникальные для каждого объекта |
| Счётчики объектов | Обычные поля (имя, возраст, и т.д.) |
| Фабричные методы | Большинство методов |
